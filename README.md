# Ultrasonic Levitator Project

## Overview
This project demonstrates acoustic levitation using 40kHz ultrasonic transducers.

## Components
- Arduino Uno
- L298N Driver
- 40kHz Ultrasonic Transducers
- Jumper Wires
- Breadboard
- 12V Power Supply

## Working Principle
Standing ultrasonic waves create pressure nodes that can suspend lightweight objects such as thermocol balls.

## Applications
- Contactless material handling
- Pharmaceutical research
- Micro-particle manipulation
- Scientific demonstrations
